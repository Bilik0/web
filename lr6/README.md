# Запуск сервера

## Щоб запустити сервер, має бути встановлений Node.js. Після цього виконуються наступні кроки:

1. Відкрити термінал у папці із файлами проєкту
2. Встановити компоненти для роботи, для цього треба ввести наступну команду:

```
npm install
```

3. Запустити сервер наступною командою:

```
node server.js
```

4. У браузері ввести посилання видане терміналом:

```
http://localhost:3000
```
